前言

用了好久pycharm最后发现这个编译器压根就不适合玩AI(可能是我不会用),但最近我发现了一个迷你的编译器spyder简直超级好用,不管玩python编程还是玩AI编程都特别好用,外加学研会的python任务所以打算给大家写个python安装快速入门

第一步:下载python

首先进入python的官网(https://www.python.org/getit/),选择Download中的windows(如下图所示)

接着找到python3.5.2版本,然后下载(如下图所示)

注意:记得是python3.x.x版本不是python2.x.x版本,因为这两个版本的代码是不相容的,这里我选择了python3.5.2是因为我进行AI编程时相对比较好用,当然你也可以选择python3.6.x或者python3.7的版本

第二步:安装python

打开刚刚下好的python安装包,按照图中所示把勾给勾上

接着直接点击Install Now直接安装,等待安装完毕(如下图所示)

安装完成后在键盘上使用”WIN+R”组合键打开运行窗口并输入cmd打开命令提示符(如下图所示)

打开命令提示符后,输入python,显示结果如下图所示表示安装成功！(如下图所示)

第三步:安装spyder

按照上面的方法打开命令提示符,输入:“pip install spyder”,然后等待下载以及安装(如下图所示)


安装完成后,在命令提示符中输入spyder3即可打开编译器,现在我们就大功告成了！！！在输入框内输入代码,然后点击spyder上方的绿色箭头就能编译运行程序,我们试试输入Helloworld(如下图所示)

现在我们就大功告成了！！！,

切记,每次打开spyder都要通过”win+R”组合键打开cmd,并输入spyder3才能打开,如果你想快一点的话就找到spyder3的快捷方式复制粘贴到桌面就好了！
